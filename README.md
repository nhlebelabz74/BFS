# BFS
C++ and Java implementations of a BFS algorithm that searches for the shortest path to the goal from the start coordinate of a maze. The maze is as follows:

14 17
xxxxxxxxxxxxxxxxx
x      x        x
x xxxx   x      x
x    xxxxx   S  x
x      x   xxxx x
x  xxx xxxxx    x
xxxxxx     x xxxx
x          x    x
x  xxxxxxxxxxxx x
x          x    x
x Gx xxxx  x xxxx
x  x       x    x
x  x            x
xxxxxxxxxxxxxxxxx
